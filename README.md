# scheme-repl

A simple repl for a small subset of the scheme language, written in haskell.

## Usage
Install cabal then:
```bash
cabal build scheme-repl
```
```bash
cabal run scheme-repl
```
## Intro haskell resources used:
- https://wespiser.com/writings/wyas/home.html

- https://en.wikibooks.org/wiki/Write_Yourself_a_Scheme_in_48_Hours
